    3. PER OGNI FILE DENTRO COLLEZIONE CREA ODM DataFile E LO PARSA IN BASE A ESTENSIONE
    4. SCRIVE OGNI ODM DELLA COLLEZIONE DENTRO UN JSON (DATA-*.JSON) (SERVIRÃ  PER LE QUERY)
    5. MANDA AL CLIENT NOMI DEI FILE

(CLIENT) CHIEDE *LOAD VIZ DATA*

(SERVER) RICEVE RICHIESTA *LOAD VIZ DATA* (PULL VIZ)
    1. LEGGE E PARSA JSON VIZ.JSON
    2. MANDA OGGETTO VIZUALIZATIONS A CLIENT (CHARTDATA E QUERY DI OGNI DS IN QUERIESBYDS ALL'INIZIO VUOTI, RIEMPITI DA ALTRE RICHIESTE FUTURE)

(CLIENT) RICEVE OGGETTO VIZUALIZATIONS
    1. LO INSERISCE NELLA VAR GLOBALE VIZUALIZATIONS
    2. CHIAMA LOADVIZGRID CHE CREA UI LINK CARD OER OGNI VIZ (QUI PROBLEMA CHE GRAFICO VUOTO PRIMA DELLA QUERY)
        - SE CHARTDATA HA GIA' DEI DATI => CREO CHART E BASTA
        - SE NO => CHIAMO loadVizQueriesResults
